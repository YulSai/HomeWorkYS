Пакет Student.additional.comparator:
ComparatorMark, ComparatorName и CompareAge - классы с реализацией интерфейса Comparator объекта Student

Пакет Student.additional.predicate:
StudentAgeGreaterAndEqual тест для возраста
StudentMarkGreater тест для оценки

Пакет Student.additional.random:
RandomForStudent - класс с реализаций методов для получения рандомных значений для объекта,
                   созданного с именем из файла

Пакет Student.additional.resources:
Name.txt - файл с именами для объектов

Пакет Student.additional.supplier:
EmptyStudentSupplier создается пустое имя
RusNameFromArraySupplier создается рандомное имя из конкретного списка
RusNameFromRandomCharacterSupplier создается имя с рандомным сочетанием букв русского алфавита
StringFromArraySupplier создается рандомное имя из списка
StudentWithNameAndIdSupplier создается объект с набором характеристик
StudentWithNameSupplier создается объект с набором характеристик без ай ди
StudentWithOutNameSupplier создается объект с набором характеристик, но без имени

Пакет Student.additional:
DataMaker - класс с методами для различных операций с обьектами обобщенного типа
Student - класс, описывающий объект
StudentListMake - класс с методами для различных операций с обьектами Student

Пакет Student.runners:
StudentMain - класс с точкой входа
